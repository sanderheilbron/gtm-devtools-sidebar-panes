Options for GTM DevTools Sidebar Panes extension

Show panes

Displays a sidebar pane in Developer Tools with an overview of all available macros inside the google_tag_manager object.

Displays a sidebar pane in Developer Tools which contains the content of the Google Tag Manager dataLayer (when available inside the DOM).

Displays a sidebar pane in Developer Tools which contains registered GTM click events. To use this pane you have to install and use the GTM Auto-Event Listener Debugger Chrome extension by Simo Ahava.
